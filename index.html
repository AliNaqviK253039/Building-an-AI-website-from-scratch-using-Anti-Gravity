Step 01

The Frontend (What Users See)

Start with just 3 simple files in a folder on your computer.

index.html 
<!-- This is your page structure -->
<div class="chat-box">
  <input id="userInput"
    placeholder="Ask me anything..." />
  <button onclick="askAI()">
    Send 🚀
  </button>
  <div id="response"></div>
</div>
💡 Beginner tip: HTML is just labels! <div> means "box", <input> means "text field", <button> means "button". That's all!
Step 02

The Backend (The Middle Layer)

Think of the backend as a waiter at a restaurant 🍽️

1 🙋

You (the customer) place an order — type a question

→
2 🧑‍🍳

Waiter (backend) takes it to the kitchen (AI)

→
3 🍱

Kitchen cooks the answer and sends it back to you

💡 Anti-Gravity shortcut: For beginners, skip building a backend! Use a service like Vercel Edge Functions or call the AI API directly from JavaScript. One less thing to carry!
Step 03

Adding AI (The Magic Layer)

An API is like a power socket 🔌 — plug in, get power.

1
Sign up at OpenAI, Anthropic, or Gemini (all have free tiers)
2
Get an API Key — it's like a password that lets your website talk to the AI
3
Send a fetch() call from JavaScript with the user's question
4
Display the answer — put the AI response into a div on your page!
💡 fetch() is just JavaScript saying: "Hey AI, here's a question!" and waiting for the reply.
